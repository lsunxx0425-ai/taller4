\section{\texorpdfstring{$|x + 3| + |x- 1| \geq 5$}{|x + 3| + |x- 1| \geq 5}}

\begin{itemize}
    \item Casos pels valors absoluts:
\end{itemize}

\[
|x + 3| = 
\begin{cases}
x + 3 & \text{si } x \geq -3 \\
- x - 3 & \text{si } x < -3
\end{cases}
\]

\[
|x - 1| = 
\begin{cases}
x - 1 & \text{si } x \geq 1 \\
-x + 1 & \text{si } x < 1
\end{cases}
\] \\

Casos a estudiar: $(-\infty, -3), [-3, 1), [1, +\infty)$.\\

$\boxed{x < -3}$ \\
$|x + 3| + |x - 1| \geq 5 \Rightarrow -x - 3 - x + 1 -5 \geq 0 \Rightarrow -2x - 7 \geq 0 \Rightarrow$ \fbox{$x \leq \frac{-7}{2}$} \\

$\boxed{-3 \leq x < 1}$ \\
$|x + 3| + |x -1| \geq 5 \Rightarrow x + 3 - x + 1 \geq 5 \Rightarrow -1 \geq 0$ IMPOSSIBLE! \\

$\boxed{ x \geq 1}$ \\
$|x + 3| + |x - 1| \geq 5 \Rightarrow x + 3 + x - 1 \geq 5 \Rightarrow 2x - 3 \geq 0 \Rightarrow$ \fbox{$x \geq \frac{3}{2}$} \\

$\textbf{Solució: }\,(-\infty, \frac{-7}{2}] \cup [\frac{3}{2}, +\infty)$
